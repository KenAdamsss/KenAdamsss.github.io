ubuntu 依赖问题
sudo apt-get update # 更新
sudo apt-get -f install # 解决依赖问题
sudo dpkg -i xxx.deb # 重新安装
卸载
先找到软件的名称：whereis,which,find,grep 都可以
可以直接使用 dpkg 命令进行卸载，这叫做谁污染谁治理；
dpkg -l |grep xxx
# xxx 就是安装的软件名

找到之后，使用下面的方法进行卸载
sudo dpkg --purge xxxx
rm -rf ~/.local/share/xxxx

这样的话就彻底卸载了
